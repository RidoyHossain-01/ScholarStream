# 🎓 ScholarStream

ScholarStream is a modern scholarship management platform that helps students discover, apply, and track scholarships easily. It provides a seamless experience for applicants and administrators with secure authentication, real-time updates, and payment integration.

---

## 🌐 Live Website
🔗 https://scholar-stream-original.web.app/

---

## 🚀 Features

### 👩‍🎓 For Students
- Browse and search scholarships
- View detailed scholarship information
- Apply for scholarships online
- Secure application fee payment
- Track application status from dashboard
- User authentication & profile management

### 🛠️ For Admin
- Add, update, and delete scholarships
- View total scholarship statistics
- Monitor applications
- Dashboard analytics with charts
- Manage users and roles
